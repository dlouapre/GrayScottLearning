# Gray-Scott Learning

Provenance of Life: Chemical Autonomous Agents Surviving through Associative Learning
by Stuart Bartlett and David Louapre

"Scenes" folder contains the two delivery condition cases considered in the paper : Uniform and Boundary Diffusion

For each scene, simulation parameters can be adjusted in the Editor Inspector window the SimulationManager object.
In particular Antidote Production controls the network : no network, reactive, pre-emptive or associative learning.
